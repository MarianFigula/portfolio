Unreadable copied text changer

In one of my courses we were getting assignments in form of PDF files written in Latex. Since files were in Slovak, copying text from these PDFs led to errors with special characters like "č", "š", "ĺ", etc. making the copied text unreadable.

I created a simple executable application to automatically change the special characters when user copies text from PDFs. This tool saved time and ensured that the text is readable.

Add workout to TickTick (Web extension)

At my university gym, students and employees must book sessions using the university website. I found it annoying to manually add these workout sessions to my TickTick app, which I use for planning.

To solve this, I created a web extension that automatically adds workout times and dates to TickTick using its API. I also added a button directly on the university website to make it simpler and more efficient.
