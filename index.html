Real world projects

Resap

Resap is a web application that allows users and companies to create and manage reservations.

I was taking care of both frontend and backend side using Javascript and PHP.

Intrasy

Intrasy is a web and mobile application allowing users to monitor their rides without the need to purchase a GPS device.

As a frontend developer my task was to develop the UI and UX of the application.

Unreadable copied text changer

When studying for my master's degree, one of my courses has assignments written in LaTeX, resulting in PDFs written in Slovak. However, copying text from these PDFs led to errors with special characters like "č", "š", "ĺ", etc. making the copied text unreadable.

I created a simple executable application to automatically change the special characters when user copies text from PDFs. This tool saved a time and ensured that the text is readable.

Add workout to TickTick (Web extension)

At my university gym, students and employees must enroll in sessions using the university website. I found it annoying to manually add these workout sessions to my TickTick app, which I use for planning.

To solve this, I created a web extension that automatically adds workout times and dates to TickTick using its API. I also added a button directly on the university website to make it simpler and more efficient.

School task System

Me and my team created a task management system for teachers and students using Laravel and Docker. Teachers can manage tasks and assign them to students, who can then view and complete them. After submission, teachers can evaluate the tasks.

Olympians management

The project was aimed at understanding authentication logic. The backend is in PHP, and Bootstrap is used for the frontend.
