# /weby screenshots

Drop the landing-page screenshots here. They are copied verbatim by Vite into
`dist/weby/assets/` on `npm run build` and served from
`https://marianfigula.com/weby/assets/`.

Required files (referenced by `src/weby/translations/*.json` and `public/sitemap.xml`):

- `lechman.png`  — screenshot of https://www.lechman-advokat.sk/
- `bellegis.png` — screenshot of https://bellegis.sk/

Recommended: ~16:10 ratio (e.g. 1280×800), optimized PNG/JPG. Until the real
files are added, the `<img>` tags on the page will 404.
