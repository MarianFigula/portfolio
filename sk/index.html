Projekty

Projekty z reálneho sveta

Resap

Resap je webová aplikácia, ktorá umožňuje používateľom a spoločnostiam vytvárať a spravovať rezervácie.

Mojou prácou bol vývoj ako na backende, tak aj na frontende, s využitím PHP a Javascriptu.

Intrasy

Intrasy je webová a mobilná aplikácia, ktorá používateľom umožňuje monitorovať ich jazdy bez potreby zakúpenia zariadenia GPS.

Ako frontend developer bolo mojou úlohou vyvíjať UI a UX webovej aplikácie.

Zmena nečítateľného skopírovaného textu

Počas inžinierskeho štúdia sme na jednom z predmetov dostávali zadania ako pdf súbory napísané v LaTeX-u v slovenčine. Avšak kopírovanie z týchto PDF súborov viedlo k chybám pri špeciálnych znakoch ako "č", "š", "ĺ" a pod., čo spôsobilo, že skopírovaný text bol nečítateľný.

Vytvoril som teda jednoduchú spustiteľnú aplikáciu, ktorá automaticky zmení a opraví špeciálne znaky pri kopírovaní textu z PDF. Aplikácia nám ušetrila čas a zabezpečila čitateľnosť textu.

Pridanie tréningu do aplikácie TickTick (webové rozšírenie)

V našej univerzitnej telocvični sa študenti a zamestnanci musia na tréningy prihlásiť prostredníctvom univerzitnej webovej stránky. Prišlo mi pracné pridávať tréningy manuálne do aplikácie TickTick, ktorú používam ako todo list.

Pre zjednodušenie som si vytvoril webové rozšírenie, ktoré automaticky pridáva tréningy do aplikácie TickTick pomocou jej API. Webové rozhranie umožňuje pridať tlačidlo priamo na univerzitnú stránku pre jednoduchšie a efektívnejšie využitie.

Školské projekty

Aplikačný server pre správu kníh

Aplikačný server umožňuje používateľom vytvárať, získavať, aktualizovať a vymazávať knihy. Server je API pre operácie CRUD.

Pri projekte som využíval Javu, Spring Boot a uplatnil som už nadobudnúte princípy OOP.

Systém školských úloh

Ja a môj tím sme vytvorili systém na správu úloh pre učiteľov a študentov využitím Laravelu a Dockeru Učitelia môžu spravovať úlohy a prideľovať ich študentom, ktorí si ich môžu prezerať a plniť. Po odoslaní môžu učitelia úlohy vyhodnotiť.

Bol som tímový líder, pracoval som na backende a pomáhal som s implementáciou Dockeru.

Manažment olympionikov

Vytvoril som webovú aplikáciu s dvoma skupinami používateľov: správcovia a používatelia. Správcovia dokážu spravovať olympionikov, zatiaľ čo používatelia si olympionikov a ich výsledky iba prezerať. Aplikácia zahŕňa prihlasovanie prostredníctvom služby Google Authenticator a Google.

Projekt bol zameraný na pochopenie logiky autentizácie. Backend je napísaný v jazyku PHP, na frontend je použítá knižnica Bootstrap.
